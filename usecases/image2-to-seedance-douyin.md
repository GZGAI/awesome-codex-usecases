# 从生图到发布抖音的组合型大工作流案例

## 案例名称

- Codex 从 gpt-image-2 生图到 Seedance 2 图生视频再到 HyperFrames 包装并发布抖音
- Codex 做 AI 生图、图生视频、视频包装、抖音发布全链路
- Codex 做从视觉生成到短视频分发的一体化工作流

## 组合的原子工作流

1. `gpt-image-2` 生图
2. `Seedance 2` 图生视频
3. `HyperFrames` 转场、配音、视频包装
4. 抖音自动发布

## 是否适合转换为 Codex 用例

非常适合，属于标准的组合型大工作流。

## 常见案例提纲

1. 明确主题与受众
2. 用 `gpt-image-2` 生成主视觉图
3. 输出 `Seedance 2` 图生视频参数与镜头节奏
4. 用 `HyperFrames` 做转场、字幕、配音、BGM 与包装
5. 生成抖音标题、简介、标签与封面文案
6. 自动上传并发布到抖音
7. 记录链接与数据反馈
